# Session Completion

The session is ending. Follow these guidelines to ensure proper completion.

## Context Preservation

Before the session ends:
- Summarize key decisions and outcomes from this session
- Note any pending tasks or unfinished work
- Document important context that should persist

## Cleanup Requirements

Ensure proper cleanup:
- Complete any in-progress operations or cancel them gracefully
- Confirm all critical changes have been persisted
- Release any held resources

## Session Summary

Generate a session summary including:
- What was accomplished during this session
- Key decisions made and their rationale
- Outstanding items requiring follow-up
- Recommendations for the next session

## Handoff Information

If another agent or session will continue this work:
- Provide clear context about current state
- List any dependencies or prerequisites
- Note any known issues or blockers
- Include relevant file paths and identifiers

## Final Status

Report the session outcome:
- **Success**: All goals achieved
- **Partial**: Some goals achieved, remaining work documented
- **Blocked**: Unable to complete due to identified issues
